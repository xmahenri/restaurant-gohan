# 🍱 Gohan Restaurant Landing Page

Uma landing page moderna e responsiva para um restaurante fictício chamado **Gohan**, inspirada na culinária japonesa. O projeto foca em UI/UX, responsividade e construção de interfaces web modernas.

🔗 **Live Demo:** https://xmahenri.github.io/gohan-restaurant-landing-page/

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (básico/interatividade)
- Git & GitHub
- GitHub Pages (deploy)

## 🎯 Objetivo do Projeto

Este projeto foi criado com o objetivo de praticar:

- Estruturação de landing pages modernas
- Organização de componentes visuais
- Responsividade para diferentes telas
- Publicação de sites usando GitHub Pages
- Boas práticas de front-end

## 🧠 Funcionalidades

- Layout totalmente responsivo (mobile, tablet e desktop)
- Seção de menu de pratos com foco visual
- Design clean e moderno inspirado na culinária japonesa
- Navegação fluida entre seções
